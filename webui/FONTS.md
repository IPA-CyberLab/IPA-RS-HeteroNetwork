# UI font

`noto-sans-jp-ui.ttf` is a glyph subset of Noto Sans JP used by the Japanese UI.
The source font is distributed by the Google Fonts project under the SIL Open
Font License 1.1. The license text is in `OFL-NotoSansJP.txt`.

Source: https://fonts.google.com/noto/specimen/Noto+Sans+JP

The subset contains the non-ASCII characters referenced by `app.js` and
`index.html`; Latin text continues to use the normal system font stack.
